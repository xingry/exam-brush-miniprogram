# 高职计算机考证刷题小程序

## 📱 项目定位
面向高职学生的计算机类专业考证刷题工具，主打免费 + 智能推送。

## 🎯 MVP 功能清单

### 已实现
- [x] 项目结构搭建
- [ ] 首页（今日任务 + 快捷入口）
- [ ] 刷题页面（答题 + 解析）
- [ ] 错题本（自动收藏）
- [ ] 个人中心（学习统计）

### 待实现
- [ ] 每日打卡
- [ ] 分享裂变
- [ ] 付费押题卷

## 🛠️ 技术栈
- 前端：微信小程序原生
- 后端：微信云开发（免运维）
- 数据库：云数据库

## 📂 目录结构
```
exam-brush/
├── app.js                 # 小程序入口
├── app.json               # 全局配置
├── app.wxss               # 全局样式
├── pages/
│   ├── index/             # 首页
│   ├── practice/          # 刷题页
│   ├── mistakes/          # 错题本
│   └── profile/           # 个人中心
├── components/            # 公共组件
├── utils/                 # 工具函数
└── cloudfunctions/        # 云函数
```

## 🚀 快速开始

1. 导入微信开发者工具
2. 配置云开发环境
3. 导入题库数据
4. 预览/上传代码
