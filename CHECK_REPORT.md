## 🐛 发现的问题

### 高优先级（影响核心功能）

1. **❌ 云数据库未集成**
   - 所有数据使用硬编码示例
   - 无法持久化存储用户数据
   - 影响功能：刷题记录、错题本、学习统计

2. **❌ 用户登录未实现**
   - 无法获取用户 openid
   - 无法区分不同用户数据
   - 云函数调用缺失

3. **❌ 题库数据缺失**
   - 目前仅 3 道示例题
   - 需要至少 500 题真实题库
   - 科目分类未完善

### 中优先级（影响用户体验）

4. **⚠️ 反馈页面不存在**
   - 个人中心点击"意见反馈"会报错
   - 需创建 feedback 页面或移除入口

5. **⚠️ 得分计算未实现**
   - 练习完成后显示固定分数
   - 应统计实际答题正确率

6. **⚠️ 错题收藏功能不完整**
   - 仅显示 toast，未实际保存
   - 需接入云数据库

### 低优先级（可后续优化）

7. **📝 项目配置占位符**
   - appid 使用 `wx0000000000000000`
   - 云环境 ID 使用 `exam-brush-env`
   - 需替换为真实值
